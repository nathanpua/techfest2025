User Configuration

To link scans to your account, you need to enter your User ID from the website.
How to find your User ID:
  1. Log in to the Expose website
  2. Go to Settings
  3. Find your User ID in the Profile section
  4. Copy and paste the entire UUID (looks like: 123e4567-e89b-12d3-a456-426614174000)

Important: The User ID must be in the correct UUID format or scans won't be saved.
Enter the URL of your Expose website to enable direct links to your scan history.
Example: https://exposeai.netlify.app or https://your-expose-website.com
